## ✨ Features

- **🌓 Dark/Light Mode Theme:** Seamlessly switch between a clean light mode and a sleek dark layout.
- **📱 Fully Responsive:** Carefully crafted to look perfect on desktops, tablets, and mobile devices.
- **✨ Smooth Animations:** Delightful micro-interactions, floating hero elements, and scroll reveal animations.
- **📖 Justified Typography:** Clean, easy-to-read, and beautifully aligned content structures.
- **⚡ Blazing Fast:** Built with Vite and React for optimal performance and rapid loading.

## 📂 Project Structure

```text
src/
├── components/       # Reusable UI sections (Hero, About, Projects, Contact, Skills)
├── hooks/            # Custom React hooks (e.g., useReveal for scroll animations)
├── index.css         # Global variables, tokens, and overarching styles
├── App.css           # Component-specific styles and animations
├── App.tsx           # Main application shell and routing
└── main.tsx          # Application entry point
```

## 🚀 Getting Started

If you'd like to run this portfolio locally or use it as inspiration, follow these simple steps:

### Prerequisites

- Node.js (v16.0.0 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ad1e/Aldwin-Portfolio.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd Aldwin-Portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Build for production:**
   ```bash
   npm run build
   ```
